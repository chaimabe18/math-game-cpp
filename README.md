# Math Game (C++)

## Overview
An interactive console-based math quiz game that generates random arithmetic questions and evaluates user answers in real-time.

## Features
- Random question generation (addition, subtraction, multiplication)
- Real-time answer validation
- User interaction through console input/output
- Simple and engaging gameplay

## Technologies
- C++

## How to Run
1. Compile:
g++ math_game.cpp -o game

2. Run:
./game

## Example
Question: 5 + 3 = ?
Your Answer: 8
Correct!
